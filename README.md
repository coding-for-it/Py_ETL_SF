# Python Snowflake ETL Pipeline


## Project Overview

This project builds an ETL pipeline using:

- Python
- Pandas
- Snowflake


## ETL Flow


CSV File

↓

Extract Data using Python

↓

Transform Data

↓

Load Into Snowflake

↓

SQL Analytics



## Run Project


Install packages:


pip install -r requirements.txt



Run pipeline:


python src/main.py



## Snowflake Table


SALES table stores cleaned business data.
